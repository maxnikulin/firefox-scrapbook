# Attempt to adapt ScrapBook X plugin for Android

Firefox plugin ScrapBook X is an open project based on ScrapBook Plus and ScrapBook.

* Whole pages or their fragments may be saved to the root ScrapBook folder.
* A plain list of saved pages is available as a home screen panel.
* Open a saved page to delete it (limitation of home screen panel).
